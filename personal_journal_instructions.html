Journal Instructions

Layout Sample

Follow this structure when you build your journal thread so each root post, weekly entry, and reply stays in the right position.

Note: The "ROOT POST" is the main thread for your journal. Each "Heading 3" is a new journal entry for a week. The "Paragraph" is the starter post for that week, and the replies are comments from classmates.

The structure should look like this in your forum thread. Note that if you are using a tool that allows you to make your own threads, your starter post IS your root.

If using Canvas, add an H4 subject with this format. If using Brightspace, edit the actual subject line to match: So-and-so's Journal - Week #Letter - Topic Here
