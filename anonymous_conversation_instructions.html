Anonymous General Class Conversation (M00)

A fully anonymous board — nobody, including the instructor and TAs, can see who posted. No points attach to it.

What It's For

  • Questions you'd rather not put your name on — "am I the only one lost here?", honest feedback about the course, anything you'd hesitate to ask out loud.
  • The same classroom conduct rules apply — anonymity is for candor, not for unkindness.
  • Need a guaranteed instructor reply? Use the Class Conversation Needing TA/Instructor Replies board instead — because this board is anonymous, Canvas hides it from the tools that track which posts still need an answer, so replies here are best-effort when we browse it.
