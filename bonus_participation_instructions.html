Bonus Participation (M00)

A general place to earn extra points for things that help the class. If you point out mistakes, find broken links, create cool tutorials — post what you did here (in case we forget).

Things That Earn Bonus Points

  • Pointing out mistakes or broken links
  • Creating useful tutorials
  • Showing up when nobody else does (like when the LMS was down)
  • Showing up the last day of class, or on a lab day
  • Volunteering to demo your work
  • Providing useful feedback
  • Perfect attendance
  • Other things that help the class

How it works: post what you've done on the board and the points get added.
